# autoBAR
Free energy simulator with Tinker software packages

# Introduction

Alchemical free energy simulation using polarizable AMOEBA and AMOEBA+ force fields.

# How to setup 

Run the following command to know 
	
```shell
python autoBAR.py
```
# How to run 

Run the following command to know 
	
```shell
python autoBAR.py -h
```
